1. 接口测试服务器地址
http://127.0.0.1:2581

/*********************************/
2. 用户登录
(POST)
URL: http://127.0.0.1:2581/role/chekuserlogin
参数(必需的)：  username 用户名
参数(必需的)：  password 密码
返回值:  200 <- 登录成功 

/*********************************/
3. 根据用户名获取用户信息
(GET)
URL: http://127.0.0.1:2581/User/getByUserName?u=13383829382
参数(必需的)：  u 用户手机号
返回用户详细数据, 类型json, 结构:
[

  }
]
